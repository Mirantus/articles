    - Как сверстать 3 колонки?
  • Три дива. У первого заданы float: left и ширина. У второго float: right и ширина. У третьего margin по ширине боковых колонок.

  • А еще?
  • <strong>
  • А еще?

    - Как с помощью javascript прочитать файл с диска?

    PHP
    - Есть массив $arr = [[‘id’ => ‘1’, ‘value’ => ‘one’], [‘id’ => 2, ‘value’ => ‘two’]...]. С помощью какой функции можно его отсортировать, чтобы value был по алфавиту?
  • array_multisort или usort

    - Как из строки 'abcdef' вытащить первые три символа?
  • substr('abcdef', 0, 3);

    MYSQL
    - Как выбрать в MySql все строки в таблице table1 с id, которых нет в таблице table2?
  • SELECT table1.* FROM table1 LEFT JOIN table2 ON table1.id=table2.id WHERE table2.id IS NULL;

    - Открыл страничку. Вместо сайта - белый экран. Ошибок нет. Как найти ошибку?
  • Убедиться, что url правильный. Посмотреть исходный код страницы в браузере. Проверить, что изменения во входном файле как-то влияют. Включить display_errors. Воспользоваться xdebug.

    - Что делать если страница медленно грузится?
  • Найти узкое место с помощью time() или xdebug. Затем попытаться оптимизировать. В крайнем случае воспользоваться кэшированием.
